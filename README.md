# 🎵 Grammy Website Data Analysis

## 📌 Overview
This project analyzes the impact of the 2022 website split between Grammy.com and RecordingAcademy.com.

## 🎯 Objective
Understand how structural changes influence user behavior and engagement.

## 📊 Key Analysis
- Traffic trends and seasonal spikes
- Mobile vs Desktop usage (73%+ mobile)
- Engagement KPIs:
  - Pages per session
  - Bounce rate
  - Time on site

## 🧪 A/B Testing
- Conversion increased by **24%**
- Statistically significant at **97% confidence level**

## 🎯 Recommendation
Maintain split-site structure and optimize Grammy.com UX.
